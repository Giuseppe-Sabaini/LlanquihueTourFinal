# Descripcion general del sistema
Este proyecto corresponde a la evaluacion final de la asignatura Programacion Orientado a Objetos, Este proyecto cuenta con todos los metodos vistos
en cada semana, Encapsulamiento, Herencia, Composicion, Polimorfismo, Interfaces, Getters y setters. el proyecto trata sobre Llanquihue Tour una agencia
de turismo, Cuenta con un sistema de Registro de Clientes y Registro de Guias Turisticos, tambien se puede calcular el sueldo total de todos los guias registrados, toda la informacion queda guardada en un archivo.txt, evitando que se pierda al cerrar el proyecto 

***

# 🧱 Estructura General del Proyecto

```text
📁 resources/
├── LlanquihueFinal.txt/  #Archivo.txt encargado de guardar la informacion de (Guias y Clientes)
📁 src/
├── data/         # Archivo con la clase (GestorDatos)
├── model/       # Archivo con las clases de dominio (Cliente, Direccion, GuiaTuristico, Persona, Registrable)
├── app/           # Archivo con la clase principal (Main)
```
